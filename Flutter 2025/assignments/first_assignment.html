Recreate these designs using only Container, Row, Column, Text, Image, Icon, and SizedBox widgets.

📦 How to Use Google Fonts in Flutter

1. Add to pubspec.yaml:

dependencies:
  google_fonts: ^6.1.0

2. Import in your Dart file:

import 'package:google_fonts/google_fonts.dart';

3. Use in Text widgets:

Text(
  'Your Text Here',
  style: GoogleFonts.poppins(fontWeight: FontWeight.bold),
)
